Run Run

Design Concept:

A school project for Foundation of Game Design.

The project started as a redesign of an antlion game from previous school team. In this game, the player controls an avatar of an ant running away from the giant antlion from the bottom.

It is a 2D Top Down survival game. The player can control the ant using keyboard keys. The player can collect power ups, and freeze the giant antlion with the limited 3 water balls per level.

This game will fit casual players with the easy to learn gameplay and relaxing music.
