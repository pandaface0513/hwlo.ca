Instream - Grid Like Social Feed

Design Concept:

This website was a side-project done with a friend. We wanted to create a new way to receive information. In nowadays, social news feed flood our devices every minute and it is hard to filter out the ones we need. So why not have an universal view to look at all your feeds in a grid layout.

The search function will help you filter out the feed by hash tags. These hash tags are user-created to the photos. InStream aims to be as user intuitive as possible, user interactions are as simple as type and photos will appear.
